STEP 6 — Download Models
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

--- Model 1: YOLOv8 (traffic light detection) ---

Download:
  python -c "from ultralytics import YOLO; YOLO('yolov8n.pt')"

This saves yolov8n.pt to the current folder. Create a models folder and move it into models/:

Create folder:
  mkdir models

Moving yolov8 into models folder:
  mv yolov8n.pt models/        (macOS/Linux)
  move yolov8n.pt models\      (Windows)

--- Model 2: Traffic Sign Detection ---

Download:
  pip install huggingface_hub
  python -c "from huggingface_hub import hf_hub_download; \
           hf_hub_download(repo_id='nezahatkorkmaz/traffic-sign-detection', \
           filename='trafic.pt', local_dir='models')"

Rename the model filename to traffic_sign:
  mv models/trafic.pt models/traffic_sign.pt  (macOS/Linux)
  move models\best.pt models\traffic_sign.pt  (Windows)

Your models/ folder should look like this when done:
  models/
    yolov8n.pt
    traffic_sign.pt
