⚙️ Commands to Setup & Run
1. Install dependencies
pip install PySide6
pip install opencv-python
pip install ultralytics
2. (Optional) Upgrade pip
python -m pip install --upgrade pip
3. Download YOLO model
python -c "from ultralytics import YOLO; YOLO('yolov8n.pt')"
4. Move model to correct folder
mkdir models
mv yolov8n.pt models/

(On Windows use move instead of mv)

5. Run the app
python app.py






1️⃣ Make sure Python is installed

Check:

python3 --version

You should see something like Python 3.14.x. If not, install Python first.

2️⃣ Create a virtual environment
python3 -m venv venv
The first venv is the module
The second venv is the folder name you’re creating (you can rename it if you want)

This creates a folder venv/ in your project.

3️⃣ Activate the virtual environment
macOS / Linux:
source venv/bin/activate
Windows (cmd):
venv\Scripts\activate
